Websites

#1 HeartCare Website

The project “HEART CARE SYSTEM” is to computerize the Front Office Management of Heart Diagnosis hospital to develop a website connected to the hospital database which is user friendly, simple, fast, and cost effective. It deals with the collection of patient’s information, diagnosis details, appointment details, doctor’s details, etc. Traditionally, it was done manually. The main function of the system is to provide the user a flexible user interface to get the information about the heart rate system, treatments and the doctor.

Technologies Used:
• HTML • Java Script • PHP • Django

#2 Sky Scraping Website

A Sky-Scraper is a JavaScript-based project which is data scraping used for extracting data from NASA websites. The goal of the project is to fetch the latest images and videos along with their information from NASA’s websites.The main objective of the Project on Sky Scraping is to fetch the details of different astronomy and space science related images featured each day, along with a brief explanation. Each day, a different image or photograph of our fascinating universe is featured, along with a brief explanation written by a professional astronomer.

Technologies Used:
• HTML • CSS • Java Script • NASA APOD API
